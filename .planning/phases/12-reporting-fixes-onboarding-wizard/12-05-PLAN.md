1. **Opening balance utility** -- Create `src/lib/onboarding/opening-balance.ts`:
   - Export `getBalanceCheck(gridState: Map<string, string>)` -- the function from RESEARCH.md:
     ```typescript
     interface BalanceCheckResult {
       totalDebits: number;
       totalCredits: number;
       difference: number;
       isBalanced: boolean;
     }
     export function getBalanceCheck(gridState: Map<string, string>): BalanceCheckResult {
       let totalDebits = 0;
       let totalCredits = 0;
       for (const [key, value] of gridState) {
         const amount = parseFloat(value) || 0;
         if (key.endsWith("-debit")) totalDebits += amount;
         if (key.endsWith("-credit")) totalCredits += amount;
       }
       const difference = Math.abs(totalDebits - totalCredits);
       return { totalDebits, totalCredits, difference, isBalanced: difference < 0.005 };
     }
     ```
   - Export `generateOpeningBalanceJE(entityId: string, gridState: Map<string, string>, jeDate: Date): Promise<{ journalEntryId: string }>` that:

     - Uses existing JE creation API pattern (POST to /api/entities/:entityId/journal-entries)
     - Returns the created JE id
