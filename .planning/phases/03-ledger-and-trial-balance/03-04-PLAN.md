**API route** (`src/app/api/entities/[entityId]/trial-balance/route.ts`):
- GET handler accepting query params: `asOfDate` (ISO date string, required), `consolidated` (boolean, optional, default false)
- Validate with Zod: asOfDate required string parseable as Date, consolidated optional boolean
- If not consolidated: call `getTrialBalance(entityId, asOfDate)` from query module
- If consolidated: get all entity IDs the user has access to (from Prisma or auth context), call `getConsolidatedTrialBalance(entityIds, asOfDate)`
- Compute totals server-side: `totalDebits = sum of all row totalDebits`, `totalCredits = sum of all row totalCredits`
- Return JSON: `{ rows: TrialBalanceRow[] | ConsolidatedTrialBalanceRow[], totalDebits, totalCredits, inBalance: boolean, consolidated: boolean }`
- `inBalance` = `Math.abs(totalDebits - totalCredits) < 0.005` (epsilon for Decimal rounding)
- Return 400 for invalid params
- Follow existing API response pattern

**Column definitions** (`src/components/trial-balance/tb-columns.tsx`):
- "use client" file exporting `trialBalanceColumns: ColumnDef<TrialBalanceRow>[]`
- Columns: Account Number | Account Name | Type | Debit Balance | Credit Balance
- Account Number: sortable via DataTableColumnHeader, default sort ascending (per Claude's discretion: account number is natural default)
- Account Name: sortable
- Type: sortable, render as Badge with color per type (e.g., blue for Asset, red for Liability, green for Equity, yellow for Income, orange for Expense)
- Debit Balance: right-aligned, `formatCurrency(totalDebits)` only when > 0. Sortable by value.
- Credit Balance: right-aligned, `formatCurrency(totalCredits)` only when > 0. Sortable by value.
- Per user decision: sortable by account number, name, type, or balance (TB-03)

**Verification Banner** (`src/components/trial-balance/tb-verification-banner.tsx`):
- "use client" component
- Props: `totalDebits: number`, `totalCredits: number`
- Per user decision: persistent banner showing balance status
- Green background with CheckCircle2 icon when in balance: "In Balance -- Total Debits: $X | Total Credits: $X"
- Red background with XCircle icon when out of balance: "Out of Balance by $X -- Total Debits: $X | Total Credits: $X"
- Use `cn()` for conditional styling. Dark mode support: `bg-green-50 dark:bg-green-950` / `bg-red-50 dark:bg-red-950`
- Difference threshold: Math.abs(totalDebits - totalCredits) < 0.005 (not floating-point equality)
- Use formatCurrency for all amounts
