     ```
     if (transaction.positionId && !transaction.accountId) {

         include: { subledgerItem: { select: { accountId: true } } },
       });
       if (position) resolvedAccountId = position.subledgerItem.accountId;
     }
     ```
   - Use `resolvedAccountId` (or `transaction.accountId`) when creating the JE line items
