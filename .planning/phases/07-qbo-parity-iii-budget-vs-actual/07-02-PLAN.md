  <name>Task 1: Budget grid page with fiscal year selector, save, copy, and CSV import</name>
  <files>src/app/(auth)/budgets/page.tsx</files>
  <action>
Create src/app/(auth)/budgets/page.tsx as a "use client" page:

**Layout structure:**
- Page title "Budgets" with subtitle showing entity name
- Toolbar row: fiscal year selector dropdown (current year default, +/- 3 years range), "Copy from [year-1]" button, CSV Import button (triggers file input), Save button (primary)
- Spreadsheet grid below toolbar

**Fiscal year selector:**
- Use a Select component (shadcn) showing fiscal years (current year +/- 3).
- On change, fetch budgets for that fiscal year via GET /api/entities/:entityId/budgets?fiscalYear=N.
- Default to current calendar year on mount.

**Grid implementation:**
- Use a plain HTML table with controlled input cells (not contentEditable, not a library).
- Rows: fetch accounts for the entity filtered to INCOME and EXPENSE types (GET /api/entities/:entityId/accounts). Group into Income section and Expense section with section header rows.
- Columns: Account Number | Account Name | [12 fiscal year month columns from getFiscalYearMonths] | Total (read-only, sum of 12 months)
- Each month cell: `<input type="text" inputMode="decimal">` with right-aligned monospace text, formatted to 2 decimal places on blur.
- Grid state: React state as a Map keyed by `${accountId}-${year}-${month}` -> string amount. Initialize from fetched budget data. Empty cells default to "".
- On focus, select all text in cell. On blur, format to 2 decimal places if valid number.
- Tab key navigates between cells naturally (inputs in a table handle this).
- Section subtotal rows (Income Total, Expense Total) are read-only, computed from grid state.

**Save button:**
- Collect all non-empty cells from grid state into budgets array: { accountId, year, month, amount }.
- PUT /api/entities/:entityId/budgets with { fiscalYear, budgets }.
- Show toast on success ("Budget saved") or error.
- Disable button when no changes (track dirty state).

**Copy from prior year:**
- Fetch GET /api/entities/:entityId/budgets?fiscalYear=(currentFY - 1).
- Map prior year budget amounts into current fiscal year month positions (same month offsets, adjusted year).
- Merge into grid state (overwrite empty cells, or if grid is empty, fill all).
- Mark grid as dirty so Save is enabled.
- Show toast: "Copied N budget amounts from [year]".

**CSV Import:**
- Hidden file input triggered by Import button.
- On file select, create FormData with csvFile, POST to /api/entities/:entityId/budgets/import.
- On success, re-fetch budgets for current fiscal year to refresh grid.
- Show toast with imported/skipped counts. If errors, show them in an alert or collapsible section.

**Empty state:**
- If no budgets exist and no accounts exist, show message directing to Chart of Accounts.
- If accounts exist but no budgets, show the grid with all empty cells and a helpful message.

**Loading state:**
- Show skeleton rows while fetching accounts and budgets.

**Styling:**
- Match existing page patterns (padding, card wrapping, header sizing).
- Grid table: border-collapse, thin borders between cells, alternating row shading for readability.
- Section headers (Income, Expenses) in bold with background color, not editable.
- Subtotal rows with font-semibold and top border.
- Total row at bottom styled same as section subtotals.
