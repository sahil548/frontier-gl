1. Run `npm install @vercel/blob` to add Vercel Blob SDK.

2. Add Attachment model to prisma/schema.prisma:
```prisma
model Attachment {
  id             String       @id @default(cuid())
  journalEntryId String
  fileName       String
  fileType       String       // MIME type: "image/png", "application/pdf", etc.
  fileSize       Int          // bytes
  url            String       // Vercel Blob URL
  uploadedBy     String
  createdAt      DateTime     @default(now())
  journalEntry   JournalEntry @relation(fields: [journalEntryId], references: [id], onDelete: Cascade)

  @@index([journalEntryId])
  @@map("attachments")
}
```
Also add `attachments Attachment[]` relation to the JournalEntry model. Also add optional `templateId String?` and `template JournalEntryTemplate? @relation(fields: [templateId], references: [id])` to JournalEntry for RECR-05 (plan 03 needs this FK). Add `generatedEntries JournalEntry[]` relation to JournalEntryTemplate.

3. Run `npx prisma db push` to sync schema.

4. Create API route at `src/app/api/entities/[entityId]/journal-entries/[journalEntryId]/attachments/route.ts`:

   **GET**: List attachments for a journal entry. Auth check, verify JE belongs to entity, query `prisma.attachment.findMany({ where: { journalEntryId }, orderBy: { createdAt: 'desc' } })`. Return array.

   **POST**: Upload attachment. Auth check. Parse FormData, get `file` field. Validate:
   - File exists
   - File type in ["application/pdf", "image/png", "image/jpeg", "image/heic"]
   - File size <= 10MB (10 * 1024 * 1024)
   Return 400 with descriptive error if validation fails.
   Upload to Vercel Blob: `put(`attachments/${entityId}/${journalEntryId}/${file.name}`, file, { access: "public", contentType: file.type })`.
   Create Prisma record with fileName, fileType, fileSize, url (from blob), uploadedBy (userId).
   Return 201 with attachment record.
   NOTE: Attachments allowed on ANY JE status (draft, approved, posted) per user decision -- do NOT check status.

   **DELETE**: Delete attachment. Auth check. Accept attachment ID as query param `?id=xxx`. Find attachment, verify it belongs to the JE. Delete from Vercel Blob using `del(attachment.url)`. Delete Prisma record. Return 200.

Use existing `successResponse`/`errorResponse` pattern. Import auth from @clerk/nextjs/server. Import prisma from @/lib/db/prisma.
